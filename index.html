    Finding HTML Elements

    • If a Collection or Node List is returned:
      (Node List - Items can only be accessed by Index Number)
      (Collection - Items can be accessed by name, id, or index number)
      • You must iterate throught the list to change anything.
      • When a parent element is changed it collapses all child elements, and changes the parent element.
    • document.getElementByID(id)
      • Always Unique will return 1 item.
      let _id = document.getElementById("first-section");
console.log(_id)
      # Output:
<div id="first-section">
            <h1>This will be the main title</h1>
        </div>
    • document.getElementsByTagName
      • Returns an HTML Collection (a live list)
      let _tagNameList = document.getElementsByTagName("ul");
    console.log(_tagNameList);
      # Output :
HTMLCollection [ul]
    • document.getElementsByClassName()
      • Returns an HTML Collection (a live list)
      let _classNameList = document.getElementsByClassName("container");
console.log(_classNameList);
      # Output:
HTMLCollection(2) [div.container, div.container]

    Changing HTML Elements

    • element.innerHTML = new html content
      • let _id = document.getElementById("first-section");
_id.innerHTML = "Hello";

let _tagNameList = document.getElementsByTagName("p");
for (let i = 0; i < _tagNameList.length; i++) {
    let e = _tagNameList[i];
    e.innerHTML = "Hola";
}

let _classList = document.getElementsByClassName("container");
for (let i = 0; i < _classList.length; i++) {
    let e = _classList[i];
    e.innerHTML = "Hi";
}

Selecting from an array of tags, then adding a class to each of them.

Situ :

I need to add a class to each H1 & H3 element of the html page, to turn them green. However, there are no classes attached to these tags.



Solu :
  1. I need to make an array of the tags I am looking for. 
    2. var Hs = ["h1", "h3"];
  2. I need to iterate through each of these tags, that are now part of an array(Node List) 
    3. for (let i = 0; i < Hs.length; i++) {
    var hs = Hs[i];
};
  3. I need to find all the tags that match the items in my array. 
    4. var h = document.querySelectorAll(hs);
  4. I need to make a function that will toggle on the classes I want to show up on the tags. 
    5. function colorRed(element) {
    element.classList.toggle("oswald-font");
    element.classList.toggle("green");
}
  5. I need to toggle a class on to each of the tags.
    6. h.forEach(colorRed);
var Hs = ["h1", "h3"];
for (let i = 0; i < Hs.length; i++) {
    var hs = Hs[i];
    var h = document.querySelectorAll(hs);

    function colorRed(element) {
element.classList.toggle("oswald-font");
element.classList.toggle("green");
    }
    h.forEach(colorRed);
}
